# VetTooth Pro 🦷

Sistema de gestão odontológica veterinária para pequenos animais e equinos.

## Funcionalidades

- 📅 Agenda e Atendimentos
- 🦷 Odontograma interativo (Canino, Felino, Equino)
- 💊 Prescrições e Exames
- 💰 Financeiro (Assinaturas, Projeções, IA Financeira)
- 📋 Prontuário completo
- 🐾 Cadastro de Pacientes e Tutores

## Banco de Dados (Supabase)

1. Crie um projeto em [supabase.com](https://supabase.com)
2. Execute `supabase/schema.sql` no SQL Editor
3. Copie a URL e anon key para `supabase/client.js`
